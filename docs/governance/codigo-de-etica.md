# Código de Ética — Evangelho Digital

## 1. Compromisso com a pessoa

Tratamos cada usuário como fim em si, nunca como meio para métricas de vaidade.

## 2. Proibições absolutas

- Dark patterns e manipulação emocional
- Vício comportamental como estratégia de retenção
- Sensacionalismo, alarmismo e polarização intencional
- Pseudociência apresentada como fato
- Venda de dados pessoais

## 3. IA responsável

- Nenhum conteúdo publicado por modelo único
- Pipeline multiagente com auditoria persistida
- Transparência sobre uso de IA na geração editorial

## 4. Inclusão e acessibilidade

- WCAG 2.2 AA como padrão mínimo
- Prioridade a usuários idosos em testes de usabilidade

## 5. Accountability

- Logs estruturados e auditoria de pipeline
- Canal de contato para reclamações e direitos LGPD
